單元一：同分母分數加法 (分母一樣，好簡單！)

概念解釋

想像一下，你和你的好朋友有兩個一模一樣的披薩，每個都切成8片。你吃了 3/8 個披薩，朋友吃了 2/8 個披薩。你們一共吃了多少個披薩呢？這就是同分母分數加法的問題！

當分數的分母一樣時，就像是我們在計算相同大小的「份數」。分母代表總共有幾等份，分子代表你擁有其中的幾份。所以，我們只要把「分子」加起來，分母保持不變，因為總等份數並沒有改變喔！

看！只要把紅色的3份和橘色的2份加起來，總共就是5份，分母還是8份喔！

步驟說明

  1. 確認分母： 檢查兩個分數的分母是不是一樣的。
  2. 分子相加： 把兩個分數的分子直接相加。
  3. 分母不變： 加完後，分母仍然保持不變。
  4. 結果化簡： 如果答案是假分數，要變成帶分數；如果可以約分，要約成最簡分數喔！

範例練習

範例一：簡單相加

問題： 1/5 + 2/5 = ?

圖形輔助解釋：想像一條被分成5等份的巧克力棒。你先拿走了1份，再拿走了2份。你總共拿走了幾份呢？

計算過程：

  • 分母都是 5，一樣！
  • 分子相加：1 + 2 = 3
  • 分母不變：所以是 3/5

最終答案：1/5 + 2/5 = 3/5

範例二：結果需化簡

問題： 3/6 + 1/6 = ?

圖形輔助解釋：一個被分成6等份的蛋糕，你吃了3份，又吃了1份。共吃了幾份？

計算過程：

  • 分母都是 6，一樣！
  • 分子相加：3 + 1 = 4
  • 分母不變：所以是 4/6
  • 化簡：4/6 可以約分喔！分子分母同除以 2，變成 2/3。

最終答案：3/6 + 1/6 = 4/6 = 2/3

範例三：結果為假分數

問題： 4/7 + 5/7 = ?

圖形輔助解釋：一共有7個格子，先塗色4格，再塗色5格，總共塗色了多少格？（會超過一條喔！）

計算過程：

  • 分母都是 7，一樣！
  • 分子相加：4 + 5 = 9
  • 分母不變：所以是 9/7
  • 化成帶分數：9 除以 7 等於 1 餘 2，所以是 1又2/7。

最終答案：4/7 + 5/7 = 9/7 = 1又2/7

單元二：異分母分數加法 (分母不同，怎麼辦？)

引入概念

哇，同分母分數加法對你來說根本是小意思！但如果現在換成：你喝了 1/2 杯牛奶，你的妹妹喝了 1/3 杯牛奶。你們一共喝了多少牛奶呢？

等等，這次分母不一樣耶！一個是2，一個是3。就像一塊披薩切兩半，另一塊切三份，每份的大小都不一樣，直接把分子加起來（1+1=2）是沒有意義的！這時候，我們就需要一個神奇的魔法：「通分」！

看，1/2比1/3大，直接加起來會很奇怪對不對？

等值分數與通分

在學習通分之前，我們要先認識「等值分數」。等值分數就是分數的數值大小雖然一樣，但是分子和分母看起來卻不同。

怎麼找到等值分數呢？ 只要把分數的分子和分母同時乘以（或除以）同一個不是0的數，就可以得到它的等值分數了！ 例如：1/2 = (1x2)/(2x2) = 2/4；1/2 = (1x3)/(2x3) = 3/6。

現在回到「通分」！通分的目的就是要把不同分母的分數，變成相同分母的等值分數，這樣我們就可以像單元一那樣輕鬆加減了！這個共同的分母，通常我們會找兩個分母的「最小公倍數 (LCM)」。

小複習：什麼是最小公倍數 (LCM)？ 就是兩個數共同的倍數中最小的那一個。 例如：2 和 3 的最小公倍數是 6 (2x1=2, 2x2=4, 2x3=6...；3x1=3, 3x2=6...) 例如：4 和 6 的最小公倍數是 12 (4, 8, 12, 16...；6, 12, 18...)

步驟說明

  1. 找出公分母： 找出兩個分數分母的最小公倍數 (LCM)，這就是新的公分母。
  2. 化為等值分數： 將每個分數都變成以這個公分母為分母的等值分數。
  3. 同分母相加： 現在它們分母一樣了！將分子相加，分母不變。
  4. 結果化簡： 最後，別忘了把答案化成最簡分數，如果是假分數要變成帶分數喔！

範例練習

範例一：分母互為倍數

問題： 1/2 + 1/4 = ?

圖形/文字解說：2和4的最小公倍數是4。所以把1/2變成以4為分母的等值分數。

計算過程：

  • 分母 2 和 4 的最小公倍數是 4。
  • 將 1/2 通分：1/2 = (1x2)/(2x2) = 2/4。
  • 現在變成 2/4 + 1/4
  • 分子相加：2 + 1 = 3
  • 分母不變：所以是 3/4。

最終答案：1/2 + 1/4 = 3/4

範例二：分母互質

問題： 1/3 + 2/5 = ?

圖形/文字解說：3和5的最小公倍數是15 (因為它們互質，直接相乘)。

計算過程：

  • 分母 3 和 5 的最小公倍數是 15。
  • 將 1/3 通分：1/3 = (1x5)/(3x5) = 5/15。
  • 將 2/5 通分：2/5 = (2x3)/(5x3) = 6/15。
  • 現在變成 5/15 + 6/15
  • 分子相加：5 + 6 = 11
  • 分母不變：所以是 11/15。

最終答案：1/3 + 2/5 = 11/15

範例三：結果為假分數需化為帶分數

問題： 3/4 + 2/3 = ?

圖形/文字解說：4和3的最小公倍數是12。

計算過程：

  • 分母 4 和 3 的最小公倍數是 12。
  • 將 3/4 通分：3/4 = (3x3)/(4x3) = 9/12。
  • 將 2/3 通分：2/3 = (2x4)/(3x4) = 8/12。
  • 現在變成 9/12 + 8/12
  • 分子相加：9 + 8 = 17
  • 分母不變：所以是 17/12。
  • 化成帶分數：17 除以 12 等於 1 餘 5，所以是 1又5/12。

最終答案：3/4 + 2/3 = 17/12 = 1又5/12

範例四：結果需化簡

問題： 1/6 + 3/8 = ?

圖形/文字解說：6和8的最小公倍數是24。

計算過程：

  • 分母 6 和 8 的最小公倍數是 24。
  • 將 1/6 通分：1/6 = (1x4)/(6x4) = 4/24。
  • 將 3/8 通分：3/8 = (3x3)/(8x3) = 9/24。
  • 現在變成 4/24 + 9/24
  • 分子相加：4 + 9 = 13
  • 分母不變：所以是 13/24。
  • 13/24 已經是最簡分數了！

最終答案：1/6 + 3/8 = 13/24

學習總結與回顧 (我是分數大師！)

重點整理

  • 同分母分數加法： 分母一樣時，直接把分子相加，分母不變。記得要化簡！
  • 異分母分數加法：
    1. 第一步：通分！ 找出兩個分母的「最小公倍數」當作新的公分母。
    2. 第二步：變身！ 把每個分數都變成新的「等值分數」。
    3. 第三步：相加！ 變成同分母後，就跟同分母分數加法一樣，分子相加，分母不變。
    4. 第四步：化簡！ 最後，將答案化成最簡分數，如果是假分數要變成帶分數。

常見錯誤提醒

❌ 小心這些錯誤喔！

  • 分母直接相加： ❌ 1/2 + 1/3 不等於 (1+1)/(2+3) = 2/5！ (這是錯的喔！)
  • 未化簡分數： 忘記把答案約分到最簡，或假分數未變成帶分數。
  • 通分時分子分母沒有同乘： 變成等值分數時，分子和分母要同時乘以相同的數喔！
  • 找錯公分母： 最小公倍數很重要！多練習找 LCM 吧！
